# polyglot

## Purpose
Storage directory for Aider Polyglot benchmark results. Each run of `experimental/bench-polyglot.sh` creates a timestamped subdirectory with per-language, per-problem pass/fail results.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `results/` | Timestamped run outputs (results JSONL files) |

## For AI Agents

### Working In This Directory
- This directory contains only output artifacts. Do not manually create or modify result files.
- Results are generated by `experimental/bench-polyglot.sh` which writes to `results/YYYYMMDD-HHMMSS/`.
- Each run directory contains a `claw/results.jsonl` with per-problem outcomes (language, problem, passed, attempts).
- Supported languages are configured via `POLYGLOT_LANGUAGES` in `env.sh` (default: python, javascript, go, rust, java, cpp).
- The `.polyglot-harness/` directory (at repo root) contains the cloned Aider benchmark repo and Python venv, and is gitignored.
